Description

Command dispatcher module. All wechat requests firstly mapped to this module and dispacther to corresponding server or module for furthur process, and reply processed contents back to user.

start/1

start(Req) -> string()

  • Req = cowboy_req:req()

All wechat requests start here. This function does the followings:

1. extract parameters from request, go to step 2 if empty, otherwise go to step 3.

2. if debug mode is on, process request, otherwise print error message and reply empty response.

3. extract params and validate signature, if passed, go to step 4, otherwise reply empty response

4. check if echostr exists, if so, such request is connectivity request and reply echostr param, otherwise process request.
